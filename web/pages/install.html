Getting Set Up

To change your presence on Discord, a small background process needs to be downloaded and ran.

  1. –Authorize YouTube RPC on Discord
  2. –Download and run the program below
  3. –The app will register itself and start automatically whenever your PC turns on
  4. –Done! Your Discord presence will adjust to whatever you're doing on YouTube
Authorize on Discord Download App

The installer only writes to your user registry and AppData. No admin rights required.
